## git fetch
リモートリポジトリの情報をローカルに取り込み、ローカルのファイルは更新しない

## git pullとは
リモーリポジトリの情報をローカルに取り込み、ローカルのファイルを更新する


## コンフリクトの解消
1. mainブランチをMergeして、ローカルの作業者が使用しているVSCode上で修正
2. Github上でコンフリクト解消
